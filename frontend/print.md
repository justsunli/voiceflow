# 吸烟与肌萎缩侧索硬化症（ALS）：文献综述

**撰写时间：** 2026年3月
**研究问题：** 吸烟是否与ALS发病风险升高相关？这一关联是否具有因果性？
**研究范围：** 同行评审的流行病学研究、系统综述、荟萃分析及孟德尔随机化研究；不限时间范围。

---

## 1. 引言

肌萎缩侧索硬化症（ALS）是一种致死性进行性神经变性疾病，主要累及上下运动神经元。其病因至今尚不明确，遗传因素与环境因素均被认为与之相关。在环境因素中，吸烟是研究最为广泛的候选因素。本综述综合分析了来自观察性流行病学、荟萃分析及因果推断（孟德尔随机化）研究的证据，并探讨了吸烟与ALS之间潜在的生物学机制。

## 3. 研究发现

### 3.1 早期观察性证据（2010年前）

早期证据结论不一致。2003年，Armon在《神经流行病学》（*Neuroepidemiology*）发表了一项循证医学综述，基于两项II类研究，将吸烟认定为散发性ALS的**唯一**达到"可能"（more likely than not）等级的外源性危险因素。这一认定具有重要意义，因为在缺乏可干预危险因素的领域中，吸烟脱颖而出。

2009年，Armon在《神经病学》（*Neurology*）发表更新综述，重新审查了自2003年以来发表的28篇文献，其中7篇符合纳入标准。更新分析将吸烟的证据等级提升至散发性ALS的**A级（"确定"）危险因素**——这是循证医学中的最高等级——支持该结论的包括3项II类研究和1项III类研究，结果一致显示风险升高。

欧洲癌症与营养前瞻性调查（EPIC）队列研究（Gallo等，2009年）纳入517,890名欧洲健康受试者（共4,591,325人年），其中118例通过死亡证明确认的ALS死亡病例。Cox风险比例模型分析显示吸烟者中年龄特异性ALS死亡率升高，并通过包年数和吸烟年限探索了剂量-反应关系。

### 3.2 系统综述与荟萃分析

**Alonso、Logroscino 与 Hernán（2010年）** — 《神经病学、神经外科与精神病学杂志》（*J Neurol Neurosurg Psychiatry*）

这项里程碑式的荟萃分析汇总了18篇文献（15项病例对照研究和5项队列研究）。采用随机效应模型，合并相对风险（RR）如下：

- **当前吸烟者 vs. 从不吸烟者：** RR = 1.28（95% CI：0.97–1.68）——升高但无统计学显著性
- **曾吸烟者 vs. 从不吸烟者：** RR = 1.12（95% CI：0.98–1.27）——边界无显著性

宽置信区间反映了研究间的异质性，各研究质量差异较大。尽管统计意义处于临界水平，效应方向始终为正向。

**Weisskopf 等（2011年）** — 《JAMA神经病学》（*JAMA Neurology*）

对五项大型美国前瞻性队列（护士健康研究、男性卫生专业人员随访研究、CPS-II营养队列、多族裔队列、NIH-AARP饮食与健康研究）进行汇总分析，各队列随访时间为7至28年不等，通过全国死亡指数（NDI）确认ALS死亡病例。每增加10年吸烟史的汇总多变量RR为 **1.09（95% CI：1.03–1.16，p=0.006）**，这是当时最强的前瞻性剂量-反应证据。值得注意的是，开始吸烟年龄越早，风险越高；但每日吸烟支数与ALS风险之间未观察到剂量-反应关系。

**最新荟萃分析 — Kim 等（2024年）** — 《烟草致病》（*Tobacco Induced Diseases*）

这项已在PROSPERO注册的系统综述与荟萃分析是迄今最为全面的研究，涵盖28项病例对照研究和4项队列研究，检索数据库包括Medline、Embase、Web of Science、Scopus和ScienceDirect（检索日期：2023年1月）。主要结果如下：


| 分析类别           | 未调整OR           | 调整后OR                 |
| -------------- | --------------- | --------------------- |
| 总体（吸烟 vs. 不吸烟） | 1.14（1.06–1.22） | 1.12（1.03–1.21）       |
| 仅限当前吸烟者        | 1.28（1.10–1.49） | 1.28（1.10–1.48）       |
| 仅限女性           | 1.20（1.10–1.30） | 1.25（1.11–1.42）       |
| 仅限男性           | 1.02（0.85–1.22） | 1.01（0.80–1.28）——无显著性 |


采用限制性三次样条函数进行剂量-反应分析，结果揭示包年数与ALS风险之间呈**倒U形（非线性）关系**——中等暴露量对应最高相对风险，而极重度长期吸烟并未成比例地进一步升高风险，可能原因在于吸烟相关疾病导致的竞争性死亡。

**性别差异：** 这一关联在女性中明显更强、更稳定，在多项研究中均有体现。假说认为，这可能与男女激素环境差异、烟草毒素代谢差异，或历史性混杂（历史上女性吸烟者是一个经过选择的特殊群体）有关。

### 3.3 因果推断：孟德尔随机化研究

孟德尔随机化（MR）利用与暴露因素（此处为吸烟行为）相关的遗传变异作为工具变量，借助受孕时等位基因的随机分配来减少混杂偏倚。目前有两项MR研究直接探讨了吸烟与ALS的关系，但结论相互矛盾：

**Zhan 与 Fang（2019年）** — 《神经病学年鉴》（*Annals of Neurology*）

基于Project MinE数据（12,577例ALS患者；23,475名对照），MR分析发现**吸烟者的ALS风险高于从不吸烟者**，为因果关系提供了证据支持。这是第一项支持因果性的大规模MR研究。

**Opie-Martin 等（2020年）** — 《神经病学、神经外科与精神病学杂志》（*J Neurol Neurosurg Psychiatry*）

该研究使用了更大规模的GWAS数据集：终身吸烟（n=463,003）、曾吸烟（n=1,232,091），以及大型ALS GWAS（20,806例患者；59,804名对照）。采用五种MR方法（逆方差加权法、MR-Egger回归、加权中位数、加权众数、简单众数）及双向分析：

- 终身吸烟：OR = **0.94（95% CI：0.74–1.19），p=0.59**——无因果信号
- 曾吸烟：OR = **1.10（95% CI：1.00–1.23），p=0.05**——边界值

作者得出结论，MR方法**未发现吸烟与ALS存在因果关系的证据**，并建议未来研究采用"当前吸烟"而非"终身吸烟"或"曾吸烟"作为表型，以更好地捕捉生物学暴露。

两项MR研究的矛盾结论，可能反映了所用吸烟表型（终身综合指数 vs. 曾吸烟 vs. 当前吸烟）、统计效能及数据集样本重叠方面的差异。

## 4. 拟议的生物学机制

以下几条通路被认为可能介导香烟烟雾成分对运动神经元的损伤：

**4.1 氧化应激**
香烟烟雾含有数千种化合物，包括自由基和活性氧（ROS）。体外研究显示，即使是低浓度的尼古丁也能在大鼠中脑细胞中诱导显著的ROS生成（Ramesh等，2007年）。氧化应激可损伤运动神经元的DNA、线粒体及蛋白质稳态——这些通路与ALS发病机制直接相关，包括Nrf2/ARE信号轴（Petri等，2012年，引自机制综述）。

**4.2 神经炎症**
香烟烟雾激活NF-κB转录因子通路，促进炎症级联反应。烟草烟雾也被证实可通过ROS介导的机制损害血脑屏障（BBB）的完整性，可能使神经毒性物质进入中枢神经系统。

**4.3 重金属神经毒性**
烟草烟雾含有镉、铅等重金属，这些物质可在中枢神经系统蓄积并发挥神经毒性作用，被认为是ALS易感性的潜在共同致病因素。

**4.4 谷氨酸兴奋性毒性**
尼古丁作用于运动神经元上的烟碱型乙酰胆碱受体（nAChRs），可能调节谷氨酸的释放，从而可能促成运动神经元的兴奋性毒性死亡——这是ALS中已被充分证实的经典发病机制之一。

**4.5 尼古丁的悖论**
值得注意的是，单独使用尼古丁在某些实验模型中表现出神经保护作用（例如，通过激活α7-nAChR/Erk1/2信号通路减轻海马细胞的氧化损伤）。这一悖论使机制解读更加复杂，也可能部分解释了为何ALS风险并不随包年数线性增长。

## 5. 讨论与综合分析


| 证据类型                   | 主要发现              | 证据强度 |
| ---------------------- | ----------------- | ---- |
| 早期循证医学综述（2010年前）       | 吸烟 = 确定的（A级）危险因素  | 中等   |
| 大型前瞻性队列汇总分析            | 每多吸烟10年，风险约升高9%   | 中等   |
| 最新荟萃分析（2024年）          | OR升高12–28%；女性中更显著 | 中等   |
| MR研究（Zhan & Fang 2019） | 支持因果性             | 初步   |
| MR研究（Opie-Martin 2020） | 不支持因果性            | 中等   |


总体来看，**流行病学关联一致，但因果性尚不明确**。效应方向在各研究中稳健一致，但效应量较小（OR在1.1–1.3之间），因果问题尚待解答。

**关键悬而未决的问题：**

1. **混杂偏倚：** 吸烟与职业暴露、饮食、体力活动等多种因素共存，这些因素可能独立影响ALS风险。
2. **MR表型不匹配：** "终身吸烟"和"曾吸烟"可能无法充分反映当前的吸烟暴露负担。未来应开展以"当前吸烟"为表型的MR研究。
3. **性别异质性：** 女性中更强的关联稳定存在，但尚无解释。值得针对激素或药代动力学假说开展专项研究。
4. **剂量-反应非线性：** 最新荟萃分析中发现的倒U形曲线不符合简单的毒理学因果模型，需要机制层面的解释。
5. **家族性/遗传性ALS：** 现有研究大多聚焦于散发性ALS。吸烟是否会改变携带ALS相关突变（SOD1、TDP-43、C9orf72）者的发病风险，目前几乎无文献涉及。

## 7. 结论

流行病学证据的整体倾向支持吸烟是散发性ALS的一个**效应适度但具有一致性的危险因素**，各荟萃分析的比值比（OR）介于1.1至1.3之间。这一关联在女性中比男性更为稳健。然而，孟德尔随机化研究在因果性问题上得出了相互矛盾的结论，生物学机制也尚未完全阐明。无论这一不确定性如何，戒烟作为公共卫生措施都是审慎可取的；而吸烟究竟是ALS的直接因果驱动因素，还是与其他危险因素相关的标志物，仍是需要优先研究的科学问题。
